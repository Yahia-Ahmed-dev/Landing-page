# CodeCraft - Professional Web Development Agency

A modern, responsive landing page for a web development agency built with HTML, CSS, and JavaScript.

## Features
- Fully responsive design
- Modern UI with gradient effects
- Interactive elements with JavaScript
- Mobile-friendly navigation
- Portfolio showcase
- Client testimonials section

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts
